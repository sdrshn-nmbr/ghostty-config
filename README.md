# Ghostty Terminal Configuration

My personal Ghostty terminal configuration.

## Installation

Copy the `config` file to your Ghostty config directory:

```bash
# macOS
cp config ~/Library/Application\ Support/com.mitchellh.ghostty/config
```

## Features

- Nord-inspired color scheme
- Transparent background with blur
- Custom keybindings for tabs and splits
- Configured for macOS with tab-style titlebar
